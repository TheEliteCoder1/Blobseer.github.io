ABOUT THE GAME

Blobseer is a free 2D platformer game with no ads and an advanced in-game level editor. Explore diverse worlds, overcome challenges, and create your own levels with the powerful editor tools.

Currently in development, Blobseer already offers:

  • 12 progressively challenging levels in the Forest Sailors world
  • A comprehensive level editor with object manipulation tools
  • Multiple power-ups and game mechanics
  • Visual effects and shaders for enhanced gameplay

Note: Some features are still in development and will be marked as "Coming Soon" in-game.

LEVEL EDITOR GUIDE

Create and Customize Your Own Levels

Creating a New Level
  1. From the Main Menu, click on the "More Fun Things" button (Building Tools icon)
  2. Click on the "Level Editor" button
  3. Give your level a name (minimum 5 characters)
  4. Enter an optional description
  5. Press the "Create Level" button (Building Tools icon)
Saving Your Level

Your level is automatically saved when you close the game or use the pause menu options. You can also manually save through the pause menu.

Accessing Existing Levels
  1. From the Main Menu, click on "My Levels" (not the Level Editor button)
  2. Browse through your created levels (5 per page)
  3. Use the action buttons:
    • Edit: Return to the editor
    • Play: Test your level
    • Delete: Remove the level (with confirmation)
